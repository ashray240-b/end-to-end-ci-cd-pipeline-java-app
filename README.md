🚀 End-to-End DevSecOps Pipeline for Java Application

This project demonstrates a complete DevSecOps CI/CD pipeline for a
Java-based web application, automating the entire workflow from code
integration to deployment.

Project Overview

The pipeline ensures that every code change goes through: - Code Quality
Analysis - Automated Build - Artifact Management - Containerization -
Security Scanning - Deployment on a Multi-node Cluster

Tech Stack

-   Git
-   Jenkins
-   Maven
-   SonarQube
-   Nexus Repository
-   Docker
-   Trivy
-   Docker Stack
-   Docker Swarm
-   Linux

CI/CD Pipeline Workflow (8 Stages)

1.  Source Code Integration
2.  Code Quality Analysis
3.  Build Stage
4.  Artifact Management
5.  Docker Image Creation
6.  Security Scanning
7.  Image Registry
8.  Deployment

Architecture

GitHub → Jenkins → SonarQube → Maven → Nexus → Docker → Trivy →
DockerHub → Docker Stack (Swarm Cluster)

Key Features

-   Fully automated CI/CD pipeline
-   Integrated DevSecOps practices
-   Multi-node cluster deployment
-   End-to-end lifecycle management
